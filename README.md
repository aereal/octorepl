# Octorepl

REPL for GitHub

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'octorepl'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install octorepl

## Usage

```
octorepl
```

Also work with GitHub Enterprise:

```
octorepl --host your-github-enterprise-host.example.com
```
